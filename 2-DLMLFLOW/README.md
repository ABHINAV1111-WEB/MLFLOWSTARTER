# 🍷 Wine Quality Prediction — PyTorch + MLflow

A complete end-to-end Machine Learning project that predicts wine quality scores using a PyTorch Artificial Neural Network (ANN), with automated hyperparameter tuning via Hyperopt and full experiment tracking via MLflow.

---

## 📌 Project Overview

This project trains a regression model on the **UCI Wine Quality dataset** (white wine) to predict a quality score (0–10) from 11 physicochemical features such as alcohol content, acidity, sulphates, and pH.

The full pipeline covers:

- Data loading and preprocessing
- PyTorch DataLoader pipeline
- Custom z-score Normalization layer
- ANN model definition with PyTorch `nn.Module`
- Manual training loop with RMSE tracking
- Bayesian hyperparameter search with Hyperopt
- Experiment tracking and model logging with MLflow
- Model loading and inference via MLflow PyFunc

## 🧱 Key Components Explained

### 1. DataLoader Pipeline

```python
BATCH_SIZE = 64

def make_loader(X, y, shuffle=False):
    X_t = torch.from_numpy(X)
    y_t = torch.from_numpy(y).unsqueeze(1)   # (N,) → (N,1) for MSELoss
    return DataLoader(TensorDataset(X_t, y_t), batch_size=BATCH_SIZE, shuffle=shuffle)

train_loader = make_loader(train_x, train_y, shuffle=True)
valid_loader = make_loader(valid_x, valid_y)
test_loader  = make_loader(test_x,  test_y)
```

Converts NumPy arrays into PyTorch mini-batches. Training data is shuffled each epoch to prevent order memorisation.

---

### 2. Custom Normalize Layer

```python
class Normalize(nn.Module):
    def __init__(self, mean, var):
        super().__init__()
        self.register_buffer("mean", torch.tensor(mean, dtype=torch.float32))
        self.register_buffer("std",  torch.tensor(np.sqrt(var + 1e-7), dtype=torch.float32))

    def forward(self, x):
        return (x - self.mean) / self.std
```

Applies z-score normalisation inside the model graph. Using `register_buffer` ensures mean and std are saved with the model and move to GPU automatically — they are fixed constants, never updated by the optimizer.

### 4. Training Loop

```python
def run_epoch(model, loader, criterion, optimizer=None):
    is_train = optimizer is not None
    model.train(is_train)

    with torch.set_grad_enabled(is_train):
        for X_batch, y_batch in loader:
            preds = model(X_batch)
            loss  = criterion(preds, y_batch)

            if is_train:
                optimizer.zero_grad()
                loss.backward()
                optimizer.step()
    return rmse
```

One function handles both training (with optimizer) and evaluation (without optimizer). Gradients are disabled during evaluation for speed and memory efficiency.

---

### 5. Hyperparameter Search

```python
space = {
    "lr":       hp.loguniform("lr", np.log(1e-5), np.log(1e-1)),
    "momentum": hp.uniform("momentum", 0.0, 1.0),
}

best = fmin(fn=objective, space=space, algo=tpe.suggest, max_evals=4)
```

Hyperopt uses the **TPE (Tree-structured Parzen Estimator)** algorithm — a Bayesian approach that learns from previous trials and focuses the search on promising regions of the hyperparameter space.

| Hyperparameter | Search Range | Distribution |
|---|---|---|
| Learning rate (`lr`) | 1e-5 to 1e-1 | Log-uniform |
| Momentum | 0.0 to 1.0 | Uniform |

The parent run logs the best trial's params and model for easy retrieval.

## 📊 Evaluation Metric

This is a **regression** problem — the model predicts a continuous score, not a category. Accuracy % does not apply here. Instead:

**RMSE (Root Mean Squared Error)** measures average prediction error in the same unit as the target (quality score):

```
RMSE = √( mean( (predicted - actual)² ) )

RMSE = 0.73 means:
  model predictions are off by ±0.73 quality points on average
  on a 0–10 scale → approximately 7.3% average error
```

**R² Score** is the closest equivalent to "accuracy" for regression:

```python
from sklearn.metrics import r2_score
r2 = r2_score(true_values, predictions)
# R² = 0.35–0.45 is typical for wine quality
# (human subjectivity makes perfect prediction impossible)
```

## ⚠️ Common Gotchas

**Always use `np.log()` bounds with `hp.loguniform`:**

```python
# ❌ Wrong — passes raw values, causes TypeError
hp.loguniform("lr", 1e-5, 1e-1)

# ✅ Correct — passes log-scale bounds
hp.loguniform("lr", np.log(1e-5), np.log(1e-1))
```

**Always use `register_buffer` for fixed tensors inside `nn.Module`:**

```python
# ❌ Wrong — gets left on CPU when model moves to GPU, lost on save
self.mean = torch.tensor(mean)

# ✅ Correct — moves with model, saved with model, never trained
self.register_buffer("mean", torch.tensor(mean))
```

**Always include `"models:/"` prefix when loading by model ID:**

```python
# ❌ Wrong — MLflow looks for a file on disk
mlflow.pyfunc.load_model("m-362df6b317844b34a0bd820820e6ca84")

# ✅ Correct — MLflow looks in the Model Registry
mlflow.pyfunc.load_model("models:/m-362df6b317844b34a0bd820820e6ca84")
```
